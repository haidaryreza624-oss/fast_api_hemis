3. POST /logout

---

**Purpose:** Logs out the user.

**Request:** POST, no body

**Failure Response (JSON):**
{
"success": false,
"message": "❌ Logout failed: <error message>"
}
**Sucess response Response (JSON):**
{
    "success": true,
    "message": null
}
